# Monitoramento de Ações (MVC)

Projeto prático de padrões de projeto (MVC e Observer) desenvolvido para a disciplina de Programação para Web da UNDB.

## Funcionalidades
- Padrão **MVC (Model-View-Controller)** para separação de responsabilidades.
- Padrão **Observer** para reatividade (atualizando a View automaticamente quando o preço da ação no Model muda).
- Desenvolvido em **TypeScript** para maior robustez e tipagem estática.

## Como Executar
1. Instale as dependências: `npm install`
2. Execute o projeto localmente: `npx ts-node src/index.ts`
